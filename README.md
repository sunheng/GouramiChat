#GouramiChat

GouramiChat is a fast n' simple chat room application where users can create their own custom room and invite anyone to join. Messaging is in real-time!

####Features:
  * Instant messaging
  * Custom chat rooms
  * Instant alert of new users
  * XSS prevention
  * Mobile friendly design
